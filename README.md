Instrucciones para la configuración y ejecución del proyecto:
Descargar el proyecto del enlace de github, abrir el proyecto con visualStudio, entrar a la carpeta, hacer las instalciones que sean necesarios (comandos.MD), tener phpMyAdmin en localhost:8080, e importar el sql que también está en el proyecto


Ejemplo de configuración del archivo .env:
# Configuración de la base de datos
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=root
DB_NAME=construccion
DB_PORT=3306
# Otras configuraciones
PORT=3000
